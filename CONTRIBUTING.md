# Contributing to Labki

Thank you for your interest in contributing to Labki! This is the runtime/user repository for deploying a Labki-powered MediaWiki instance.

## Reporting Issues

When reporting problems, please include:
- Steps to reproduce
- Expected vs actual behavior
- Output of `docker compose logs wiki`
- Your OS and Docker version

## Suggesting Improvements

For feature requests or documentation improvements:
1. Check existing [Issues](https://github.com/labki-org/labki/issues) first
2. Open a new issue with a clear description

## Pull Requests

1. Fork and clone the repository
2. Make your changes
3. Test with `docker compose up -d`
4. Submit a PR with a clear description

## Platform vs Runtime

> **Note**: This repository is for the *runtime distribution*. For changes to the core platform (extensions, Docker image, MediaWiki config), contribute to [labki-platform](https://github.com/labki-org/labki-platform) instead.

## Questions?

Open a [Discussion](https://github.com/labki-org/labki/discussions) for questions or ideas.
